记 2017.3.18

March 18 2017

抱紧你的我，比国王富有，曾多么快乐。
失去你的我，比乞丐落魄，痛多么深刻。

– 《国王与乞丐》 杨宗纬 / 华晨与

国王没当多久，便成了乞丐，我的世界就像崩塌了一样，我尚未见到永远有多远，却发觉天堂与地狱只有一步之遥。

工作室的招新、优校云的优化、老师给的新项目。。。我完全没有心思去管，昨晚看到你发的微信我彻夜难眠。

我不知道要怎么才能挽留你。

脑海里惟有乘上 703 去找你的念头。

我在贡茶，五点走。

12点31分，珍珠奶盖绿茶少糖，即便你劝我回去，我仍希望我能等到你。

快到五点时，整个广州下起了暴雨。

这是老天爷想我别走，再等久点吗？

抑或是老天爷对我的嘲讽？

还是老天爷想淋醒我？

703 载着鞋子湿透的我回去了。

在车上，我终究是止不住情绪，抽泣了起来，这是自我懂事以来哭的最惨的一次。

这么久了，你应该也明白我是不会放弃的。

我不怨你，亦不恨你，我永远永远爱你，sy。
