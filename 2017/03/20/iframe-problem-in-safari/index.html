Details

  • 以前的业务流程：
    用户阅读完备注页后，点击按钮，通过

location.href = ‘itms-service://xxxxxxx’;

var ifr = document.createElement(‘iframe’);
ifr.src = ‘itms-service://xxxxxxx’;
ifr.style.display = ‘none’;
document.body.appendChild(ifr);

var link = document.createElement(‘a’);
link.href = ‘itms-service://xxxxxxx’;
link.style = “visibility: hidden”;
document.body.appendChild(link);
link.click();
对于这种用a``` 标签打开 itms-service://xxxxxxx 的方式，三个版本的 Safari 的浏览器行为终于一样了，都是能弹出下载提示，真是皆大欢喜啊，终于能回宿舍洗澡睡觉了😭😭
